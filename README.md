MTM Equipment Control System
Описание проекта

MTM Equipment Control System — это настольное приложение для учета оборудования, проведения инспекций и формирования отчетов.
Программа разработана для автоматизации контроля состояния оборудования на предприятии.

Основная цель системы — обеспечить централизованное хранение информации об оборудовании, контролировать результаты его проверки и предоставлять удобные инструменты для анализа данных.

Приложение реализовано на платформе .NET (WPF) с использованием архитектурного шаблона MVVM и ORM-библиотеки Entity Framework Core для работы с базой данных.

Основные возможности системы

Программа предоставляет следующий функционал:

1 Управление подразделениями

Пользователь с ролью администратора может:

создавать новые подразделения

редактировать существующие подразделения

удалять подразделения

просматривать список подразделений

Каждое оборудование в системе привязано к конкретному подразделению.

2 Управление оборудованием

Система позволяет:

добавлять новое оборудование

редактировать информацию об оборудовании

удалять оборудование

просматривать список оборудования

фильтровать оборудование по подразделениям

Для каждого оборудования хранится следующая информация:

название

инвентарный номер

статус оборудования

подразделение

история инспекций

3 Проведение инспекций

Инспекции используются для контроля состояния оборудования.

При проведении инспекции пользователь указывает:

дату проверки

оборудование

наличие дефекта

комментарий инспектора

Каждая инспекция автоматически привязывается к пользователю, который её создал.

4 Управление пользователями

Администратор системы может:

создавать новых пользователей

редактировать учетные записи

назначать роли

удалять пользователей

Система поддерживает разграничение прав доступа.

Роли пользователей

В системе реализовано три уровня доступа:

Администратор (Admin)

Администратор имеет полный доступ к системе:

управление подразделениями

управление оборудованием

управление пользователями

проведение инспекций

просмотр отчетов

Инспектор (Inspector)

Инспектор может:

просматривать оборудование

проводить инспекции

просматривать отчеты

Инспектор не может управлять пользователями и подразделениями.

Наблюдатель (Viewer)

Наблюдатель имеет ограниченный доступ:

просмотр оборудования

просмотр отчетов

Редактирование данных недоступно.

Отчеты

Система предоставляет возможность формирования отчетов.

Отчеты включают:

общее количество оборудования

количество проведенных инспекций

количество выявленных дефектов

статистику по оборудованию

Также доступна фильтрация отчетов:

по дате

по подразделению

Архитектура проекта

Проект разделен на несколько логических частей:

Models

Содержит классы сущностей базы данных:

User

Department

Equipment

Inspection

ViewModels

Содержит бизнес-логику приложения и команды интерфейса.

Примеры:

MainViewModel

EquipmentViewModel

UserViewModel

InspectionViewModel

ReportsViewModel

Views

Содержит пользовательский интерфейс приложения:

PasswordHasher (хеширование паролей)

  Запуск проекта

Для запуска проекта необходимо:

Установить .NET SDK

Установить SQL Server

Создать базу данных

Выполнить миграции Entity Framework

Запустить приложение

Пример команд для миграции:
```
Add-Migration InitialCreate
Update-Database
```
Тестовые учетные записи

Для входа в систему можно использовать тестовую учетную запись администратора:
```
Логин: admin
Пароль: admin
```
После входа администратор может создавать новых пользователей.
